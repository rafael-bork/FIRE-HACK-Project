title: PT-FireSprd_v3.0
description: The Portuguese Large Wildfire Spread Database (PT-FireSprd v3.0) contains detailed reconstructions of the spread of more than 155 large wildfires that occurred across Portugal between 2015 and 2025. It provides a comprehensive set of variables describing fire behaviour, historical meteorological conditions, terrain characteristics, fuel properties, and land use patterns.
